    Frequently Asked Questions

    Is this app serious?

    Absolutely not. It's a parody of productivity apps and a commentary on AI-generated code quality.

    Can I actually use this?

    Technically yes, but we recommend against relying on it for anything important. It's designed to be unreliable.

    Why would anyone make this?

    Because programming should be fun, and sometimes the best way to learn is by creating something intentionally bad.

    Are the bugs real?

    Yes! Each "feature" (bug) was carefully implemented to demonstrate common programming mistakes in an exaggerated way.
